# KielSafe
KielSafe macht dich sicherer unterwegs in Kiel!.
Die App warnt dich automatisch, wenn du dich Unfall-Hotsports näherst.
Mit GPS-basierten Geotracking, interaktiver Karte und Community-Daten hilft dir KielSafe dir Unfälle zu vermeiden und dich bewusster im Alltag zu bewegen.

Bleib sicher - fahre bewusst. 
Lade KielSafe jetzt kostenlos im Appstore herunter!
